SARVAM DENTAL CLINIC - FEATURE CODENAMES
=======================================

1) WASafeConfirm
Meaning:
- WhatsApp-dependent workflow where state changes only after user confirms `Message Sent`.

Used for:
- approve / reject with notify
- Book for Patient notify
- conflict-driven patient cancellation before block
- customer requested cancellation during unblock

2) SlotGuard
Meaning:
- slot conflict protection before allowing booking or slot-changing actions.

Used for:
- manual booking conflict checks
- already-booked slot detection
- blocked slot/day conflict messaging

3) BlockConflict
Meaning:
- emergency session blocking conflict flow when approved appointments already exist in chosen slots.

Used for:
- warning popup
- affected patient listing
- proceed-anyway path with guarded notification

4) DayOverlay
Meaning:
- centered yellow overlay card shown over blurred slot or snapshot grid when a full day is unavailable.

Used for:
- customer weekly off display
- customer blocked day display
- admin Day Snapshot weekly off / blocked day display

5) UnblockReasonFlow
Meaning:
- fixed unblock reasons instead of free text, with conditional workflow depending on reason.

Used for:
- customer didnt showup
- customer requested cancellation

6) LiveDeskAlert
Meaning:
- realtime or polling admin alert with sound for new bookings while dashboard is open.

Used for:
- bottom-left alert
- sound alert
- auto refresh prompt behavior

7) BookingWindowControl
Meaning:
- clinic settings logic controlling booking visibility and slot rules.

Used for:
- slot duration
- effective date
- next N days visibility
- permanent weekly off

8) AdminReturnLink
Meaning:
- public-page `Back to Dashboard` shortcut shown only to logged-in dentist/staff sessions.

Used for:
- recovery when dentist/staff accidentally click out of admin
- public pages only
- hidden from customers

9) ControlPanelHistoryGuard
Meaning:
- history-state pattern that closes Control Panel on Back before navigating away from dashboard.

Used for:
- mobile browser Back
- laptop/browser Back when Control Panel is open

10) InsightBoard
Meaning:
- dedicated admin analytics page with infographic-style operational summaries.

Used for:
- analytics.html
- bookings trend
- approval funnel
- service mix
- peak time slots
- weekday demand
- visits vs bookings
- blocked capacity
